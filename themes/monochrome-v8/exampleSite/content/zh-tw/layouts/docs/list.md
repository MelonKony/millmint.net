List layout 是預設產生 section 時會套用的樣式. Monochrome 提供一些參數調整它的行為。

## 設定

### List posts

你可以透過宣告 `list_posts` 來透過 `_index.md` 的內容完全客製化列表的樣式。

```yaml
list_posts: false
```

> default: true
